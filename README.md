To install dependencies:

```bash
npm install
```

To authenticate

```
.env.example -> .env
```

To run:

```bash
npm run dev
```

To run docker

```bash
docker compose build
```

```bash
docker compose --env-file .env config
```

```bash
docker compose --env-file .env up
```
